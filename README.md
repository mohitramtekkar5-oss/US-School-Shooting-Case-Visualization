# U.S. K-12 School Shooting Intelligence Dashboard

## 📊 Project Overview
This project provides a comprehensive analysis of school shooting incidents in the U.S. from 1970 to 2022. The goal was to transform complex, sensitive data into actionable intelligence to identify patterns and temporal trends.

## 🛠️ Tech Stack
* **Tool:** Power BI Desktop
* **Data Cleaning:** Power Query (M Language)
* **Analytics:** DAX (Data Analysis Expressions)
* **Visualization:** Interactive Maps, Trend Lines, and KPI Cards

## 🔍 Key Features & Insights
* **Temporal Analysis:** Visualized the rise in incidents over decades, identifying peak years and seasonal trends.
* **Geographic Mapping:** Used map visuals to identify high-frequency regions and states.
* **Custom KPIs:** Created dynamic measures for total casualties, incident counts, and year-over-year changes.
* **Interactive Filters:** Enabled users to slice data by school level (Primary vs. Secondary), shooter profile, and weapon type.
